# Portfolio - Ilias Hamlaoui

Portfolio website hosted on Vercel.

## Features
- Responsive Design
- Multi-language support
- Contact Form via Node.js Serverless Function

## Deployment
This project is set up for automatic deployment on Vercel.
The contact form uses a Node.js API in `/api/contact.js`.
